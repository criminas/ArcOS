# ArcOS

ArcOS is a modular Windows optimization framework designed to streamline, refine, and enhance a clean Windows installation without modifying or redistributing Windows itself.

ArcOS applies structured system-level configuration changes through PowerShell automation while prioritizing stability, reversibility, and Windows Update compatibility.

---

## Important

ArcOS does NOT:

- Replace Windows
- Distribute modified Windows ISOs
- Patch or tamper with Windows system binaries

ArcOS runs on top of an official, licensed Windows installation.

---

## Features

ArcOS currently includes:

- Telemetry policy adjustments
- Optional AppX package removal
- Service optimization
- Scheduled task tuning
- UI performance optimizations
- Wallpaper customization
- Power plan optimization
- Modular configuration system
- Uninstall script

All features are toggleable via `config.json`.
